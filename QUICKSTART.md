# Guia de Início Rápido: Tessy

Bem-vindo ao **Tessy**. Siga estes passos para começar em menos de 5 minutos:

### 1. Configure sua API Key
Certifique-se de que a variável `process.env.API_KEY` contenha sua chave válida do Google Gemini.

### 2. Primeiro Comando
No campo de entrada na parte inferior, digite algo como:
> "Crie um script em Python para ler um CSV e gerar um gráfico de barras"

Clique em **Transmitir** ou pressione `Ctrl+Enter`.

### 3. Ajuste os Controles
No painel direito, ative o **Tom Profissional** e coloque o **Nível de Detalhe** em 5 para uma resposta mais completa.

### 4. Use o Grounding
Pergunte sobre algo recente, como:
> "Quais as últimas notícias sobre o modelo Gemini 3?"

O ícone **BUSCA ATIVA** deve pulsar indicando que a Tessy está consultando a web em tempo real.

### 5. Salve e Exportar
Gostou do resultado? Clique em **Salvar** para guardar na Biblioteca ou **Exportar > PDF** para gerar um documento pronto para envio.

---
**Dica**: Use o botão **Templates** para ver exemplos de como estruturar seus comandos!
